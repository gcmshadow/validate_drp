#################
lsst.validate.drp
#################

The ``lsst.validate.drp`` module provides utilities for measuring and plotting Key Performance Metrics (KPMs) on processed LSST data.

.. _lsst.validate.drp-contributing:

Contributing
============

``lsst.validate.drp`` is developed at https://github.com/lsst/validate_drp.
You can find Jira issues for this module under the `validate_drp <https://jira.lsstcorp.org/issues/?jql=project%20%3D%20DM%20AND%20component%20%3D%20validate_drp>`_ component.
